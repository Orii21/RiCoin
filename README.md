# RiCoin

Ricoin adalah aplikasi crypo price

# Fitur
- Login dan Registrasi: Pengguna dapat membuat akun baru atau masuk menggunakan akun yang sudah ada. Data pengguna disimpan dengan aman menggunakan SharedPreference.
- Crypto Price: Aplikasi menyediakan informasi berupa harga terbaru dari berbagai macam coin yang ada di cryptocurrency
# Penggunaan
- Registrasi: Buat akun baru dengan mengisi formulir registrasi.
- Login: Masuk ke aplikasi menggunakan akun yang sudah terdaftar.
- Coin Detail: Pilih Coin yang ingin dilihat detail dari coin tersebut
- Search Coin: ketik nama coin yang ingin kalian cari untuk melihat detail harga dari coin tersebut
- SharedPreference : digunakan untuk menyimpan data pengguna dan informasi terkait aplikasi lainnya.
- Retrofit : untuk pengambilan API
- Androdi Studio : untuk pengembangan Aplikasi
Lottie : Untuk progressbar
